Overview

Executes a command on the shell and returns the output Optionally, provide your custom logger to logg executions

Executes a command on the shell and returns the output
class MyLogger < Logger
  def debug(logline : String)
    puts logline
  end
end

Schlib::Command.new(MyLogger.new).run("git flog")

you can also signify that you want an error to be raised if your command execution failes Schlib::Command.new.run("git flog", raise_errors: true)
